# chip-8-emu


A simple Chip-8 emulator in C++ 



## Run
When running, enter the name of the ROM file (e.g., `pong.ch8`).

## Controls
- 1, 2, 3, 4 — buttons 0x1, 0x2, 0x3, 0xC
- Q, W, E, R — 0x4, 0x5, 0x6, 0xD
- A, S, D, F — 0x7, 0x8, 0x9, 0xE
- Z, X, C, V — 0xA, 0x0, 0xB, 0xF

## Implemented
- All 35 Chip-8 instructions
- 4KB memory
- 16 registers
- Stack
- Timers
- Keyboard input
